# PCASS Metric released.
This is the code of the implementation of the  Perception-and-Cognition-inspired quality Assessment for Sonar image Super-resolution (PCASS). 

# Requirment
Matlab; 
Python.

# Usage
## Run SematicFeature.py to extract high-level features of test image and put three *.mat files into root directory
## Run the Prediction_demo.m
Note: modify the true path of your test image and check the feature data location of the test image in *.mat file
